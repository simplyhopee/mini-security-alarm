# Mini Security Alarm

This project is a simple Arduino-based security alarm designed to simulate a basic intrusion detection system. The system uses a push button as a trigger sensor and activates an LED and buzzer when the alarm is triggered.

## Features
- Detects a simulated intrusion using a push button
- Activates an LED as a visual warning
- Sounds a buzzer as an audible alert
- Demonstrates basic embedded systems programming and circuit design

## Components Used
- Arduino Uno
- Push Button
- LED
- Piezo Buzzer
- Breadboard
- Resistors
- Jumper Wires

## Skills Demonstrated
- Arduino Programming (C++)
- Digital Input and Output
- Embedded Systems Fundamentals
- Circuit Design
- Troubleshooting and Testing
